# VoxelMorph - PyTorch Implementation

PyTorch implementation of deformable image registration with support for 2D and 3D volumes.

## Implementations

- **[voxelmorph2d/](voxelmorph2d/)** - 2D registration using MNIST dataset
- **[voxelmorph3d/](voxelmorph3d/)** - 3D registration using OASIS dataset

## Datasets

- **2D**: MNIST
- **3D**: [Neurite OASIS Dataset](https://github.com/adalca/medical-datasets/blob/master/neurite-oasis.md)
