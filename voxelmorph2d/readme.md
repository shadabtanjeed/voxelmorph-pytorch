# Get Started

- Install dependencies: `pip install -r requirements.txt`
- Download and preprocess the MNIST dataset (affine alignment) using `mnist.ipynb`.
- Train the model with `train.py`. Provide the path to the processed MNIST dataset, for example: `python train.py --data_path data/mnist_normalized_32x32.npz`.
- To visualize results, run `inference.ipynb`.
